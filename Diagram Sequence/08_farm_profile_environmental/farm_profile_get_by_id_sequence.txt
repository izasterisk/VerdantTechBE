title Get Farm Profile By ID Sequence Diagram

participant Client
participant FarmProfileController
participant FarmProfileService
participant FarmProfileRepository
participant Database

Client->FarmProfileController: 1. GET /api/FarmProfile/{id}
activate FarmProfileController

FarmProfileController->FarmProfileService: 2. GetFarmProfileByFarmIdAsync(id, ct)
activate FarmProfileService

FarmProfileService->FarmProfileRepository: 3. GetFarmProfileWithRelationByFarmIdAsync(id, useNoTracking: true)
activate FarmProfileRepository
FarmProfileRepository->Database: 4. Query farm profile with Address, User, Crops relations
activate Database
Database-->FarmProfileRepository: 5. Return farm profile or null
deactivate Database
FarmProfileRepository-->FarmProfileService: 6. Return farmProfile or null
deactivate FarmProfileRepository

FarmProfileService->FarmProfileService: 7. Map to FarmProfileResponseDTO
FarmProfileService-->FarmProfileController: 8. Return FarmProfileResponseDTO or null
deactivate FarmProfileService

FarmProfileController->Client: 9. Return 200 OK (FarmProfileResponseDTO)
deactivate FarmProfileController

note over Client,Database: Returns farm profile with Address, User, and Crops relations\nNo 404 handling - returns null if not found
